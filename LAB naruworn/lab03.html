type ใน attribute: input คุณลักษณะ
1.button ปุ่มกด
2.checkbox ช่องสำหรับเช็คถูกสามารถเลือกหลายหัวข้อได้
3.color เลือกสี
4.date เลือก date หลังจากเลือกจะระบุเป็น mm/dd/yyyy
5.datetime-local เลือก datetime-local หลังจากเลือกจะระบุเป็น mm/dd/yyyy:เวลา
6.e-mail เป็นช่องกรอกสำหรับ e-mail
7.file จะมีปุ่มาสำหรับเลือกไฟล์
8.hidden เป็นพื้นที่ว่างเปล่า
9.Image รูปภาพสำหรับคลิก
10.month เลือก datetime-local หลังจากเลือกจะระบุเป็น month
11.number เป็นช่องสำหรับปรับค่าขึ้นลงของตัวเลข
12.password ช่องกรอกและแสดงผล ● เมื่อกรอกข้อมูลลงไป
13.radio ช่องสำหรับเช็ค ● และเปลี่ยนจุดสีดำใหม่เหมือน radio ที่อีกตำแหน่ง
14.range จะมีเส้นยาวไว้สำหรับ drag ไปมา
15.reset จะเป็นปุ่มสำหรับการ reset input type ทั้งหมด
16.search จะเป็นช่องสำหรับกรอกคล้าย text แต่จะมีปุ่ม x หากต้องการลบข้อมูลในช่อง search ออก
17.submit ปุ่มกด submit
18.tel เป็นช่องกรอกสำหรับเบอร์โทร
19.text เป็นช่องกรอกสำหรับข้อความ
20.time เลือก time หลังจากเลือกจะระบุเป็น เวลา pm/am
21.url เป็นช่องกรอกวำหรับ url
22.week เลือก week หลังจากเลือกจะระบุเป็น Week จำนวนสัปดาห์ใน 1 ปี, ปีที่เลือก
