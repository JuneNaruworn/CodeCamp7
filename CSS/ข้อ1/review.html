การบ้าน CSS ข้อ 1

CSS
คำถาม คำตอบ
1. CSS syntax ประกอบด้วยอะไรบ้าง รูปแบบของคำสั่ง CSS นั้นจะประกอบไปด้วย 3 ส่วนคือ
1.Selector คือ ส่วน HTML tag ที่เราต้องการจะให้แสดงผล เช่น table ก็จะแสดงผลในส่วนของตาราง
2.Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร , ชนิดตัวอักษร , ขนาดตัวอักษร
3.Value คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีแดง , ตัวอักษรแบบ tahoma , ตัวอักษรขนาด 12 pt
2. การเขียน CSS มีกี่แบบอะไรบ้าง มีอยู่ 3 รูปแบบ ดังนี้
1. Inline Style Sheet
2. Embed Style Sheet
3. External Style Sheet
3. selector มีกี่แบบอะไรบ้าง Selector คือ HTML elements หรือ elements ที่ CSS Rule ถูกนำไปใช้ ประกอบด้วยแบบต่างๆ ดังนี้
  • IDใช้เวลาในการหาน้อยที่สุด
  • Classใช้เวลาในการหามากกว่า ID เล็กน้อย
  • Tagใช้เวลาในการหามากกว่า Class
  • Universalใช้เวลาในการหามาก

    • Descendant combinators
